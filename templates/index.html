Projetos

API - Radar Cidadão

Problema: Dados públicos sobre os deputados candidatos à reeleição muito complexos para compreensão da parte majoritária da sociedade

Objetivo:Facilitar o acesso à informações difíceis de compreender sobre os candidatos à reeleição

Solução: Criação de um site que transforma dados em informações úteis e simples para a população compreender e escolher entre os candidatos

Tecnologias: PYTHON, HTML, CSS, COLAB, JIRA, BOOTSTRAP

Metodologia:Foi utilizada a metodologia ágil SCRUM

Desafios Técnicos:A fonte utilizada não possui boa parte dos dados que precisávamos para fornecer as informações desejadas ao projeto

Resultados:O projeto ainda está em desenvolvimento. Entretando, já foi possível mapear e colocar em prática algumas formas de transformar os dados em informações úteis
